## 工具和實用程式

**skill-creator Skill：**

- 內建於 Claude.ai 並可用於 Claude Code
- 可從描述生成 Skill
- 審查並提供建議
- 使用方式：「幫我使用 skill-creator 建構一個 Skill」

**驗證：**

- skill-creator 可以評估你的 Skill
- 詢問：「審查此 Skill 並建議改進」
